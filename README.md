# WAV / 808ILIAS
SOUND AS ARCHIVE.  
BLACK / OFF-WHITE.  
DEV-TERMINAL ENERGY.  

## WHAT THIS IS
AN AUDIO‑REACTIVE BEAT ARCHIVE + VISUALIZER.  
BUILT TO FEEL LIKE SYSTEM UI.   

## HOW TO USE
- CLICK **ENTER** → TERMINAL BOOTS → VISUALIZER FADES IN.
- PICK TRACKS FROM THE FILESYSTEM SIDEBAR.
- DROP YOUR OWN AUDIO WITH **+ YOUR OWN**.
- COLOR PANEL = PRIMARY + ACCENT. CUSTOM COLOR MODE ON.

## ADD YOUR BEATS
THE APP LOOKS FOR FILES IN:
```
public/beats/
```
MATCH THE FILENAMES USED IN `src/data/CatalogData.ts`.

## DEV
```
npm install
npm run dev
```

## BUILD
```
npm run build
npm run preview
```

## NOTE
BEATS ARE NOT COMMITTED BY DEFAULT (`public/beats/` IS GIT‑IGNORED).
USE YOUR OWN STORAGE OR COMMIT LOCALLY IF YOU WANT THEM DEPLOYED.
